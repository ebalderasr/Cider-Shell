# Extension Manager Guide

This guide uses both English and Spanish names because GNOME translations vary by package version.

## 1. User Themes / Temas de usuario

Install and enable:

- `User Themes`
- `Temas de usuario`

After enabling it, open `GNOME Tweaks` or `Retoques` and set:

- `Shell`: `WhiteSur-Light`

## 2. Just Perfection

Open the extension settings and use values close to these:

- `Activities Button` / `Boton Actividades`: Off
- `App Menu` / `Menu de aplicaciones`: Off
- `Search` / `Buscar`: On
- `Workspace Popup` / `Ventanas emergentes del espacio de trabajo`: Off
- `Window Picker Icon` / `Icono del selector de ventanas`: Off
- `Clock Menu Position` / `Posicion del reloj`: Center
- `Panel Size` / `Tamano del panel`: 30 or 32
- `Startup Status` / `Estado al iniciar`: Desktop
- `Accessibility Indicator` / `Indicador de accesibilidad`: Off

If the exact translation is different, look for the keywords:

- `Activities` or `Actividades`
- `Clock` or `Reloj`
- `Panel`
- `Workspace` or `Espacio de trabajo`
- `Accessibility` or `Accesibilidad`

## 3. Blur my Shell

Suggested settings:

- `Top Panel Blur` / `Desenfoque del panel superior`: On
- `Dash Blur` or `Dock Blur` / `Desenfoque del dash` o `Desenfoque del dock`: On
- `Overview Blur` / `Desenfoque de la vista general`: On
- medium blur strength
- medium opacity

Avoid very strong blur or it starts to look artificial.

## 4. Ubuntu Dock

If you prefer to confirm the dock visually in Settings:

- `Settings` > `Ubuntu Dock`
- `Configuracion` > `Ubuntu Dock`

Recommended values:

- `Position on screen` / `Posicion en pantalla`: `Bottom` / `Inferior`
- `Panel mode` or `Extend to screen edge` / `Modo panel`: Off
- `Auto-hide the Dock` / `Ocultar automaticamente el dock`: On
- `Icon size` / `Tamano de iconos`: 48 or 56
- `Show mounted drives` / `Mostrar unidades montadas`: Off
- `Show Trash` / `Mostrar papelera`: Off

## 5. Tweaks / Retoques

In `Appearance` / `Apariencia`:

- `Applications` / `Aplicaciones`: `WhiteSur-Light`
- `Icons` / `Iconos`: `WhiteSur`
- `Cursor`: `WhiteSur-cursors`
- `Shell`: `WhiteSur-Light`

In `Window Titlebars` / `Barra de titulo de ventanas`:

- `Maximize` / `Maximizar`: On
- `Minimize` / `Minimizar`: On
- `Placement` or `Button Layout` / `Colocacion de botones`: Left
