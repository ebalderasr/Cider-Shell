# Visual Finishing

Use this guide after `install.sh` if you want the desktop to feel more convincingly Mac-like.

## English

Recommended finishing touches:

- Set a clean light wallpaper with soft gradients or landscape photography
- Keep the dock at `48` or `56` icons, not larger
- Use `GNOME Tweaks` to confirm `WhiteSur-Light` is selected for both `Applications` and `Shell`
- Keep top bar clutter low: no seconds, no weekday, no accessibility icon
- Use `Blur my Shell` with medium blur only
- Keep favorites minimal: browser, files, terminal, settings, app store
- Reboot after theme changes if the shell theme does not refresh immediately

Optional upgrades:

- If you already own or installed `SF Pro`, switch fonts manually for a closer macOS feel
- Add a curated wallpaper pack in this repo later under `assets/wallpapers/`
- Add before/after screenshots to the README

## Espanol

Usa esta guia despues de `install.sh` si quieres que el escritorio se sienta mas cercano a macOS.

Ajustes recomendados:

- usa un fondo claro y limpio, con degradados suaves o fotografia de paisaje
- deja el dock en `48` o `56`, no mas grande
- confirma en `Retoques` que `WhiteSur-Light` este activo en `Aplicaciones` y `Shell`
- manten la barra superior limpia: sin segundos, sin dia de la semana y sin icono de accesibilidad
- usa `Blur my Shell` con desenfoque medio, no exagerado
- deja pocos favoritos en el dock: navegador, archivos, terminal, configuracion y tienda
- reinicia sesion si el tema del shell no cambia de inmediato

Mejoras opcionales:

- si ya tienes `SF Pro`, cambiala manualmente para un look mas cercano a macOS
- puedes agregar un paquete de wallpapers en el repo bajo `assets/wallpapers/`
- puedes agregar capturas de antes y despues al README
